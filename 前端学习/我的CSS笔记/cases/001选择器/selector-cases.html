CSS3-选择器练习案例
E:nth-child(n)

描述

选择属于其父元素的第n个子元素的每个 E 元素。

代码示例

指定每个 p 元素匹配的父元素中第 2 个子元素的背景色：

<style> .case-nthChild h1:nth-child(2){ background:#ff0000; } </style> <div class="case-nthChild"> <h1>这是一个标题</h1> <p>这是第一个段落。</p> <p>这是第二个段落。</p> <p>这是第三个段落。</p> <p>这是第四个段落。</p> </div>

这是第四个段落。

E:nth-of-type(n)

描述

选择属于其父元素第n个 E 元素的每个 E 元素。

代码示例

指定每个p元素匹配同类型中的第2个同级兄弟元素的背景色：

.case-nthoFType p:nth-of-type(2) { background:#ff0000; } <div class="case-nthoFType"> <h1>这是一个标题</h1> <p>这是第一个段落。</p> <p>这是第二个段落。</p> <p>这是第三个段落。</p> <p>这是第四个段落。</p> </div>
